# SilkStep

**Course:** 01VRP — Introduction to Web Applications (2025/2026)

**Student:** Mo Mokashmiri — S308968

## About

SilkStep is a free walking tours platform inspired by the historic Silk Road routes through Iran. The project starts in **Tehran**, with themed walks across the city’s old quarters, bazaars, and landmarks.

**Tagline:** *Every step tells a story.*

## Tech stack

- HTML5, CSS3, Bootstrap
- Flask + SQLite
- Flask-Login
